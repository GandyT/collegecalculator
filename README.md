# CollegeCalculator
![gh commits](https://img.shields.io/github/last-commit/GandyT/collegecalculator)

calculate your chances of getting into a school based on prior years

### Run the Project

```sh
python -m venv venv
venv\Scripts\activate
pip install requirements.txt
python main.py
```
